About Ovaxon

Ovaxon is a simple website that lets you look up a movie and see which streaming services it is available on.

What this site uses

  • HTML and CSS for structure and design
  • JavaScript to search movies and update the page
  • The TMDB API for streaming provider data

Why it was built

This site was created for WDD 131 to practice basic HTML, CSS, JavaScript, and working with an API.
