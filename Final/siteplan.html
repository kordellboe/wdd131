Overview

Purpose

The purpose of this site is to be a simple web version of my Ovaxon app. Users will be able to look up a movie and see where it is currently streaming based on data stored with the TMDB database.

Audience

The audience is movie fans who have one or more streaming services and want a quick way to check where they can watch a specific movie without jumping between different apps.

Dynamic elements

Dynamic behavior will be handled with JavaScript:

  • Store a list of movies as an array of objects. Each movie will include title, year, genre, and a list of streaming services.
  • A search bar that filters the movie list by title as the user types.
  • A streaming service filter that shows only movies available on the selected service.
  • A button that picks a random movie from the list and shows it as a suggestion to watch.

Wireframes

Home page (index.html)

Sections on the page:

  • Header with logo and simple navigation.
  • Search area with a text input, streaming service dropdown, and a search button.
  • Results area where movie cards are generated by JavaScript. Each card shows title, year, genre, and where it is streaming.
  • Random suggestion area with a button that picks a movie from the list and displays it.
  • Footer with small text and link to the child page.

Child page (about or help page)

This page will be simpler and mostly static:

  • Short explanation of how Ovaxon works.
  • Example screenshot or image of the app or website.
  • FAQ style text like "How do you get streaming data" and "What services are included".
